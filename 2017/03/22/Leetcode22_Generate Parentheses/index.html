Leetcode22_Generate Parentheses

Veröffentlicht am 2017-03-22 | in Leetcode . 算法 | |
  • 所谓Backtracking是这样的思路，在当前局面下，你有若干种选择；那么尝试每一种选择，如果已经发现某种选择肯定不行（违反了某些限定条件），就返回；如果某种选择到最后发现是正解，就将其加入解集
  • 所以思考递归题时，只要明确三点：选择（Options）、限制（Restrains）、结束条件（Termination）
  • 对于这道题
    • 任何时候有两种选择
      • 加左括号
      • 加右括号
    • 限制条件
      • 左括号用完了，就不能再用左括号了
      • 右括号不能比左括号多
    • 结束条件
      • 左右括号都用完了
