Two Sum

Veröffentlicht am 2017-02-08 | in Leetcode | |

Given an array of integers, return indices of the two numbers such that they add up to a specific target.You may assume that each input would have exactly one solution.

总结

  • Map是一种将一种兑现映射到其他对象的强大工具
  • 本题思路
    • 将数组映射到map,注意将nums[i]映射为Key,这样就可以利用map.get(Object key)得到数组下标
    • map.containsKey(Object key)能够得到key对应的value–本题的数组下标
  • HashMap常用函数：
    • Map.containsKey(Object key)–找key值对应的value值
    • Map.containsValue(Object value)–找一个或多个key值对应的value
    • Map.get(key) 返回key对应的value值
    • Map.put(key,value) 插入
    • Map.size() 返回映射数
    • Map.remove(key) 移除key值对应的键值对
