补充：StringBuffer类

在线帮助文档

  • 线程安全的可变字符序列，一个类似于String的字符串缓冲区，但不能修改
  • StringBuffer上的主要操作是append和insert方法，可接受任意类型的数据
  • 例如:
    • z引用的当前内容为start，调用z.append(“le”)会使字符串缓冲区包含“startle”
    • z.insert(4,le),将更改字符串缓冲区，使它包含 “starlet”
  • 通常，如果z引用当前StringBuilder的一个实例，sb.append(x)和sb.insert(sb.length(),x)具有相同效果
  • 从JDK5开始，为该类补充了一个单线程使用的等价类，即StringBuilder。与该类相比，通常优先使用StringBuilder类，因为它支持所有相同的操作，但是不执行同步，所以速度更快

常用方法：append()、insert()、delete()、reverse()等

  • append(char c)
  • append(char[] str)
  • append(String str)
  • append(char[] str , int offset , int len)
  • capacity()
  • charAt()
  • insert(int offset , char[] str)
  • lastIndexof(String str)
  • toString()
