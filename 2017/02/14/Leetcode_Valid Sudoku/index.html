Valid Sudoku

Posted on 2017-02-14 | In Leetcode | |

思路

  • 碰到判断一个容器中是否有相同元素时，都可以考虑用HashSet来解决
  • 此题中一共需要创建9个Rows、9Cols、9Cubes容器，所以要将new放进第一层for循环中
  • 利用HashSet中的add(),如果含有此元素就返回false，不存在就放进容器，好用

小结:

  • 遍历3*3立方体board[3][3]
  • 遍历9*9立方体

可以粗糙地认为，%比/增长得快，%放在Rows就是先将行遍历完，放在列就是先将列遍历完
